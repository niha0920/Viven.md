## File Management & Disk Management in Operating System
### 1. File Management
**File management** is a function of the operating system that handles creation, deletion, storage, retrieval, naming, and organization of files.
#### What is a File?
A file is a collection of related information stored on secondary storage devices like hard disks or SSDs.

Examples:
* Text file (`notes.txt`)
* Image file (`photo.jpg`)
* Program file (`program.c`)
* Video file (`movie.mp4`)
#### Objectives of File Management
* Store data permanently
* Organize files efficiently
* Allow easy access and retrieval
* Protect files from unauthorized access
* Manage file sharing
#### Functions of File Management
1. **File Creation**

   * Creating new files.
2. **File Deletion**

   * Removing files when no longer needed.
3. **File Access**

   * Reading, writing, and modifying file contents.
4. **File Organization**

   * Organizing files in folders/directories.
5. **File Protection**

   * Providing permissions (read, write, execute).
6. **Backup and Recovery**

   * Recovering deleted or corrupted files.
#### File Attributes
A file contains information such as:
* File name
* File type
* Size
* Creation date
* Owner
* Permissions
* Location
#### File Operations
Common operations:
* Create
* Open
* Read
* Write
* Rename
* Copy
* Move
* Delete
* Close
#### Directory Structure
A directory is used to organize files.

Types:
1. **Single-Level Directory**

   * All files in one directory.
2. **Two-Level Directory**

   * Separate directory for each user.
3. **Tree-Structured Directory**

   * Hierarchical organization.
4. **Graph Directory Structure**

   * Shared files/folders.

Example:
```
Root
│
├── Documents
│   ├── notes.txt
│   └── report.doc
│
├── Images
│   └── photo.jpg
│
└── Videos
    └── movie.mp4
```
#### Advantages of File Management
* Efficient storage
* Easy searching
* Better security
* Data organization
* Data sharing
#### Disadvantages
* Complexity in management
* Storage overhead
* Security risks if not managed properly
## 2. Disk Management
**Disk management** is the process of managing disk storage, allocation, organization, and maintenance.

It helps the OS efficiently use storage space.
### Objectives of Disk Management
* Efficient storage utilization
* Faster data access
* Reduce fragmentation
* Improve system performance
### Functions of Disk Management
#### 1. Disk Formatting
Preparing a disk for use.

Types:
* **Low-Level Formatting**
  * Divides disk into sectors and tracks.
* **High-Level Formatting**
  * Creates file system (FAT, NTFS, EXT4).
#### 2. Disk Partitioning
Dividing disk into multiple sections called partitions.

Example:
* C: Drive → OS
* D: Drive → Personal files
#### 3. Free Space Management
Tracks free and used storage blocks.

Methods:
* Bit vector
* Linked list
* Grouping
* Counting
#### 4. Disk Scheduling
Determines order of disk access requests to improve performance.

Common algorithms:
* **FCFS (First Come First Serve)**
* **SSTF (Shortest Seek Time First)**
* **SCAN**
* **C-SCAN**
* **LOOK**
* **C-LOOK**
#### 5. Disk Allocation Methods
How files are stored on disk.
1. **Contiguous Allocation**
   * Stores file in continuous blocks.
2. **Linked Allocation**
   * Blocks linked together.
3. **Indexed Allocation**
   * Uses index block to store addresses.
### Disk Structure
Disk contains:
* **Platter** → Circular disk
* **Track** → Circular path
* **Sector** → Small storage unit
* **Cylinder** → Group of tracks
### Fragmentation
Fragmentation occurs when files are scattered in memory.

Types:
1. **Internal Fragmentation**
   * Unused space inside allocated block.
2. **External Fragmentation**
   * Free memory scattered in different locations.
### Defragmentation
Process of rearranging fragmented files to improve performance.
### Advantages of Disk Management
* Better performance
* Efficient storage utilization
* Faster access
* Reduced data loss
### Disadvantages
* Complex implementation
* Maintenance required
* Fragmentation issues
## Difference Between File Management and Disk Management
| File Management              | Disk Management                |
| ---------------------------- | ------------------------------ |
| Manages files and folders    | Manages disk storage           |
| Focuses on file organization | Focuses on storage efficiency  |
| Handles file access          | Handles disk allocation        |
| Example: create/delete files | Example: partition/format disk |
