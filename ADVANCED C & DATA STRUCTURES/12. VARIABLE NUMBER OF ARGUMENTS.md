# Implementation of printf()
`printf()` is a standard library function used to print formatted output to the screen.

A simple custom implementation can be done using:
* `stdarg.h` → for variable arguments
* `putchar()` → to print characters one by one
## Basic Implementation of printf()
```c
#include <stdio.h>
#include <stdarg.h>
void my_printf(const char *format, ...)
{
    va_list args;
    va_start(args, format);
    while(*format)
    {
        if(*format == '%')
        {
            format++;
            switch(*format)
            {
                case 'd':
                {
                    int num = va_arg(args, int);
                    printf("%d", num);
                    break;
                }
                case 'c':
                {
                    char ch = va_arg(args, int);
                    putchar(ch);
                    break;
                }
                case 's':
                {
                    char *str = va_arg(args, char *);
                    while(*str)
                    {
                        putchar(*str);
                        str++;
                    }
                    break;
                }
                default:
                    putchar('%');
                    putchar(*format);
            }
        }
        else
        {
            putchar(*format);
        }
        format++;
    }
    va_end(args);
}
int main()
{
    my_printf("Number = %d\n", 100);
    my_printf("Character = %c\n", 'A');
    my_printf("String = %s\n", "Hello");
    return 0;
}
```
## Output
```
Number = 100
Character = A
String = Hello
```
## Important Concepts Used
### 1. Variable Argument Functions
`printf()` accepts variable number of arguments.

Example:
```c
printf("%d %s", 10, "Hello");
```
To handle this:
* `va_list`
* `va_start`
* `va_arg`
* `va_end`
are used from `stdarg.h`.
## Flow of Execution
```
my_printf("Value = %d", 10)
↓
format string scanned character by character
↓
'%' found
↓
next character checked ('d')
↓
va_arg() fetches integer argument
↓
integer printed
```
## Meaning of stdarg Macros
| Macro        | Purpose                  |
| ------------ | ------------------------ |
| `va_list`    | Holds argument list      |
| `va_start()` | Starts reading arguments |
| `va_arg()`   | Fetches next argument    |
| `va_end()`   | Cleans up                |
## Real printf() Internally
Actual `printf()`:
* Parses format specifiers
* Converts numbers to strings
* Handles width/precision
* Supports:
  * `%d`
  * `%f`
  * `%x`
  * `%p`
  * `%u`
  * `%lld`
  * etc.
* Finally calls low-level `write()` system call
## Time Complexity
If string length is `n`:
```
O(n)
```
because format string is scanned once.
## Very Important Internal Working
Actual flow:
```
printf()
   ↓
vprintf()
   ↓
vsnprintf()
   ↓
write() system call
   ↓
stdout
```
## Difference Between printf and sprintf
| Function     | Output Destination |
| ------------ | ------------------ |
| `printf()`   | Console            |
| `fprintf()`  | File               |
| `sprintf()`  | String buffer      |
| `snprintf()` | Limited buffer     |

# Implementation of scanf()
`scanf()` is used to take formatted input from the user.

A custom implementation can be created using:
* `stdarg.h` → variable arguments
* `getchar()` → reading input character by character
## Basic Implementation of scanf()
```c
#include <stdio.h>
#include <stdarg.h>
void my_scanf(const char *format, ...)
{
    va_list args;
    va_start(args, format);
    while(*format)
    {
        if(*format == '%')
        {
            format++;
            switch(*format)
            {
                case 'd':
                {
                    int *num = va_arg(args, int *);
                    scanf("%d", num);
                    break;
                }
                case 'c':
                {
                    char *ch = va_arg(args, char *);
                    *ch = getchar();
                    break;
                }
                case 's':
                {
                    char *str = va_arg(args, char *);
                    scanf("%s", str);
                    break;
                }
                default:
                    break;
            }
        }
        format++;
    }
    va_end(args);
}
int main()
{
    int num;
    char ch;
    char str[100];
    my_scanf("%d", &num);
    my_scanf("%c", &ch);
    my_scanf("%s", str);
    printf("Number = %d\n", num);
    printf("Character = %c\n", ch);
    printf("String = %s\n", str);
    return 0;
}
```
## Output
```
100
A
Hello

Number = 100
Character = A
String = Hello
```
## Important Concept
Unlike `printf()`, `scanf()` stores values into variables.

So we pass:
* addresses (`&num`)
* pointers internally
Example:
```c
scanf("%d", &num);
```
`scanf()` modifies memory using pointer.

## Internal Working of scanf()
```
scanf("%d", &num)
↓
Reads input from stdin
↓
Converts ASCII string to integer
↓
Stores result at given memory address
```
## Why Pointer is Needed?
Because function must modify original variable.

Example:
```c
int x;
scanf("%d", &x);
```
Without address:
* only copy is passed
* original variable cannot change

| Macro        | Purpose                   |
| ------------ | ------------------------- |
| `va_list`    | Holds argument list       |
| `va_start()` | Starts variable arguments |
| `va_arg()`   | Gets next argument        |
| `va_end()`   | Cleanup                   |

# Advanced Manual scanf() Implementation
Without using actual `scanf()`:
```c
#include <stdio.h>
void input_int(int *num)
{
    int n = 0;
    char ch;
    while((ch = getchar()) != '\n')
    {
        n = n * 10 + (ch - '0');
    }
    *num = n;
}
int main()
{
    int x;
    input_int(&x);
    printf("Number = %d\n", x);
    return 0;
}
```
# Explanation
Input:
```
123
```
Processing:
```
n = 0

'1' → 0*10 + 1 = 1
'2' → 1*10 + 2 = 12
'3' → 12*10 + 3 = 123
```
# ASCII Conversion Logic
Character digit to integer:
```c
'ch' - '0'
```
Example:
```
'5' - '0'
53 - 48 = 5
```
## Real scanf() Internally
Actual `scanf()`:
* reads buffer from stdin
* parses format string
* converts ASCII to required datatype
* handles:
  * width
  * whitespace
  * floating point
  * hexadecimal
  * error checking
Internally uses:
```
vscanf()
↓
vfscanf()
↓
read() system call
```
## Common scanf() Problems
## 1. Buffer Overflow
```c
char str[5];
scanf("%s", str);
```
Input longer than 5 causes overflow.

Safer:
```c
scanf("%4s", str);
```
## 2. scanf with %c Issue
```c
scanf("%c", &ch);
```
reads newline from previous input.

Solution:
```c
scanf(" %c", &ch);
```
(space before `%c` skips whitespace)
## Time Complexity
If input length is `n`:
```
O(n)
```
because characters are scanned once.
