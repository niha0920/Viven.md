# In Ascii format
## Writing Structure To File
```c
#include<stdio.h>
struct Student
{
    int id;
    char name[20];
    float marks;
};
int main()
{
    FILE *fp;
    struct Student s1 = {101, "Niharika", 95.5};
    fp = fopen("student.txt", "w");
    fprintf(fp, "%d %s %f", s1.id, s1.name, s1.marks);
    fclose(fp);
    return 0;
}
```
### Reading Structure From File
```c
#include<stdio.h>
struct Student
{
    int id;
    char name[20];
    float marks;
};
int main()
{
    FILE *fp;
    struct Student s1;
    fp = fopen("student.txt", "r");
    fscanf(fp, "%d %s %f", &s1.id, s1.name, &s1.marks);
    printf("ID = %d\n", s1.id);
    printf("Name = %s\n", s1.name);
    printf("Marks = %f\n", s1.marks);
    fclose(fp);
    return 0;
}
```
### ASCII File Content
```
101 Niharika 95.500000
```
### Explanation
* `fprintf()` → Writes structure data into file in ASCII/text format.
* `fscanf()` → Reads structure data from file.
* `"w"` → Write mode.
* `"r"` → Read mode.
* Data is stored as readable text in ASCII format.

# In Binary format
## Writing Structure To Binary File
```c 
#include<stdio.h>
struct Student
{
    int id;
    char name[20];
    float marks;
};
int main()
{
    FILE *fp;
    struct Student s1 = {101, "Niharika", 95.5};
    fp = fopen("student.dat", "wb");
    fwrite(&s1, sizeof(struct Student), 1, fp);
    fclose(fp);
    return 0;
}
```
### Reading Structure From Binary File
```c 
#include<stdio.h>
struct Student
{
    int id;
    char name[20];
    float marks;
};
int main()
{
    FILE *fp;
    struct Student s1;
    fp = fopen("student.dat", "rb");
    fread(&s1, sizeof(struct Student), 1, fp);
    printf("ID = %d\n", s1.id);
    printf("Name = %s\n", s1.name);
    printf("Marks = %f\n", s1.marks);
    fclose(fp);
    return 0;
}
```
### Explanation
* `fwrite()` → Writes structure directly in binary format.
* `fread()` → Reads structure from binary file.
* `"wb"` → Write binary mode.
* `"rb"` → Read binary mode.
* Binary files are not human readable.
* Binary format is faster and uses less memory compared to ASCII format.
### Function Syntax
```c 
fwrite(&structure_variable, sizeof(structure), 1, file_pointer);
fread(&structure_variable, sizeof(structure), 1, file_pointer);
```

# Modifying a structure in the file
## Program To Modify Structure Record In Binary File
```c 
#include<stdio.h>
struct Student
{
    int id;
    char name[20];
    float marks;
};
int main()
{
    FILE *fp;
    struct Student s;
    fp = fopen("student.dat", "rb+");
    while(fread(&s, sizeof(struct Student), 1, fp))
    {
        if(s.id == 101)
        {
            s.marks = 99.0;
            fseek(fp, -sizeof(struct Student), SEEK_CUR);
            fwrite(&s, sizeof(struct Student), 1, fp);
            break;
        }
    }
    fclose(fp);
    return 0;
}
```
### Explanation
* `"rb+"` → Opens binary file for both reading and writing.
* `fread()` → Reads structure record from file.
* Condition checks required record using `id`.
* `fseek()` moves file pointer back to current structure position.
* `fwrite()` overwrites old record with modified data.
### Important Function
```c 
fseek(fp, -sizeof(struct Student), SEEK_CUR);
```
### Meaning
* `fp` → File pointer
* `-sizeof(struct Student)` → Move backward by one structure size
* `SEEK_CUR` → From current position
### File Operation Flow
```
Read Record -> Check Record -> Move Pointer Back ->
Modify Data -> Write Updated Record
```
