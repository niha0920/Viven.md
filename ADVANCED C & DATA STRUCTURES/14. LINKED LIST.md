# Introduction Linked List
A **Linked List** is a linear data structure where elements are stored in separate memory locations and connected using pointers.

Unlike arrays, linked list elements are **not stored in contiguous memory locations**.

Each element in a linked list is called a **node**.

A node contains:
1. **Data** – actual value
2. **Pointer (Link)** – address of the next node
### Structure of a Node
```c
struct Node
{
    int data;
    struct Node *next;
};
```
### Representation of Linked List
```
[10 | * ] → [20 | * ] → [30 | NULL]
```
* `10`, `20`, `30` are data values
* `*` stores the address of the next node
* `NULL` indicates the end of the list
### Basic Components
#### 1. Head Pointer
* Stores the address of the first node
* Used to access the entire linked list
```c
struct Node *head = NULL;
```
## Features of Linked List
* Dynamic size
* Easy insertion and deletion
* Memory allocated during runtime
* No memory wastage like fixed arrays
## Advantages
1. Dynamic memory allocation
2. Efficient insertion/deletion
3. No need for contiguous memory
4. Flexible data structure
## Disadvantages
1. Extra memory required for pointers
2. Sequential access only
3. Searching is slower than arrays
4. Complex pointer handling
## Types of Linked Lists
### 1. Singly Linked List
Each node points to the next node.
```
10 → 20 → 30 → NULL
```
## 2. Doubly Linked List
Each node contains:
* Previous pointer
* Next pointer
```
NULL ← 10 ⇄ 20 ⇄ 30 → NULL
```
## 3. Circular Linked List
Last node points back to the first node.
```
10 → 20 → 30
↑         ↓
← ← ← ← ←
```
## Operations on Linked List
1. Creation
2. Traversal
3. Insertion
4. Deletion
5. Searching
6. Updating
7. Reversing
## Simple Linked List Program in C
```c
#include<stdio.h>
#include<stdlib.h>
struct Node
{
    int data;
    struct Node *next;
};
int main()
{
    struct Node *head = NULL;
    struct Node *second = NULL;
    struct Node *third = NULL;
    head = (struct Node*)malloc(sizeof(struct Node));
    second = (struct Node*)malloc(sizeof(struct Node));
    third = (struct Node*)malloc(sizeof(struct Node));
    head->data = 10;
    head->next = second;
    second->data = 20;
    second->next = third;
    third->data = 30;
    third->next = NULL;
    struct Node *temp = head;
    while(temp != NULL)
    {
        printf("%d ", temp->data);
        temp = temp->next;
    }
    return 0;
}
```
### Output
```
10 20 30
```

# Advantages of Linked List over an Array
A **Linked List** has several advantages compared to an **Array**, especially when dynamic memory management and frequent insertions/deletions are required.
## 1. Dynamic Size
* Arrays have a fixed size.
* Linked lists can grow or shrink during runtime.
### Array
```c
int arr[5];
```
Size is fixed to 5.
### Linked List
Nodes can be added anytime using dynamic memory allocation.
## 2. Efficient Insertion and Deletion
* In arrays, insertion/deletion requires shifting elements.
* In linked lists, only pointer changes are needed.
### Array Insertion
```
10 20 30 40
Insert 25 at position 3

Shift elements → Time consuming
```
### Linked List Insertion
```
10 → 20 → 30

Insert 25:
10 → 20 → 25 → 30
```
Only links are updated.
## 3. No Memory Wastage Due to Preallocation
* Arrays may allocate extra unused memory.
* Linked lists allocate memory only when needed.
```c
malloc(sizeof(struct Node));
```
Memory is created dynamically.
## 4. No Need for Contiguous Memory
* Arrays require continuous memory locations.
* Linked lists use scattered memory locations connected through pointers.

This makes memory allocation easier in low-memory conditions.
## 5. Better Memory Utilization
Memory is used only for existing nodes.

Example:
* Array size = 100
* Only 20 elements used
* Remaining 80 locations wasted

Linked list creates only 20 nodes.
## 6. Easy Implementation of Dynamic Data Structures
Linked lists are widely used to implement:
* Stacks
* Queues
* Graphs
* Hash tables
* Polynomial manipulation
## 7. Flexible Expansion
No need to create a larger structure and copy elements like arrays.

Array resizing requires:
1. Create new array
2. Copy old elements
3. Delete old array

Linked list simply adds a new node.
## Comparison Table
| Feature            | Array      | Linked List    |
| ------------------ | ---------- | -------------- |
| Size               | Fixed      | Dynamic        |
| Memory Allocation  | Static     | Dynamic        |
| Insertion/Deletion | Slow       | Fast           |
| Memory Requirement | Contiguous | Non-contiguous |
| Memory Wastage     | Possible   | Less           |
| Access Speed       | Fast       | Slower         |
## Main Advantage
The biggest advantage of a linked list is:

> **Efficient insertion and deletion with dynamic memory allocation.**
## Real-Time Example
### Music Playlist
Songs can be:
* Added
* Removed
* Rearranged dynamically

A linked list is more suitable than an array because playlist size changes frequently.

# Operation on Singly Linked Lists
A **Singly Linked List** is a collection of nodes where each node contains:
1. Data
2. Pointer to the next node
```
[Data | Next] → [Data | Next] → NULL
```
## Basic Operations on Singly Linked List
1. Creation
2. Traversal
3. Insertion
4. Deletion
5. Searching
6. Updating
7. Counting Nodes
8. Reversing
## Structure of Node
```c
struct Node
{
    int data;
    struct Node *next;
};
```
## 1. Creation of Linked List
Creating nodes dynamically using `malloc()`.
### Example
```c
struct Node *newnode;
newnode = (struct Node*)malloc(sizeof(struct Node));
```
## 2. Traversal
Traversal means visiting each node one by one.
### Algorithm
1. Start from head
2. Print node data
3. Move to next node
4. Repeat until NULL
### Program
```c
void display(struct Node *head)
{
    struct Node *temp = head;
    while(temp != NULL)
    {
        printf("%d ", temp->data);
        temp = temp->next;
    }
}
```
### Output
```
10 20 30
```
## 3. Insertion Operations
Insertion can be done:
1. At beginning
2. At end
3. At specific position
## A) Insertion at Beginning
### Steps
1. Create new node
2. Store data
3. Point new node to head
4. Move head to new node
### Program
```c
void insertBeginning(struct Node **head, int value)
{
    struct Node *newnode;
    newnode = (struct Node*)malloc(sizeof(struct Node));
    newnode->data = value;
    newnode->next = *head;
    *head = newnode;
}
```
### Before
```
10 → 20 → 30
```
### After inserting 5
```
5 → 10 → 20 → 30
```
## B) Insertion at End
### Steps
1. Create new node
2. Traverse to last node
3. Link last node to new node
### Program
```c
void insertEnd(struct Node *head, int value)
{
    struct Node *newnode, *temp;
    newnode = (struct Node*)malloc(sizeof(struct Node));
    newnode->data = value;
    newnode->next = NULL;
    temp = head;
    while(temp->next != NULL)
    {
        temp = temp->next;
    }
    temp->next = newnode;
}
```
## C) Insertion at Specific Position
### Steps
1. Traverse to desired position
2. Adjust links
### Example
```
10 → 20 → 30

Insert 25 after 20

10 → 20 → 25 → 30
```
## 4. Deletion Operations
Deletion can be done:
1. From beginning
2. From end
3. From specific position
## A) Deletion from Beginning
### Steps
1. Store head node
2. Move head to next node
3. Free old node
### Program
```c
void deleteBeginning(struct Node **head)
{
    struct Node *temp;
    temp = *head;
    *head = (*head)->next;
    free(temp);
}
```
## B) Deletion from End
### Steps
1. Traverse to second last node
2. Remove last node
3. Set next as NULL
## C) Deletion at Specific Position
Adjust links and free the selected node.
## 5. Searching
Finding a particular element in linked list.
### Program
```c
void search(struct Node *head, int key)
{
    int pos = 1;
    while(head != NULL)
    {
        if(head->data == key)
        {
            printf("Element found at position %d", pos);
            return;
        }
        head = head->next;
        pos++;
    }
    printf("Element not found");
}
```
## 6. Updating Node Data
Changing existing node value.
### Example
```
10 → 20 → 30

Update 20 to 25

10 → 25 → 30
```
## 7. Counting Number of Nodes
### Program
```c
int countNodes(struct Node *head)
{
    int count = 0;
    while(head != NULL)
    {
        count++;
        head = head->next;
    }
    return count;
}
```
## 8. Reversing a Linked List
### Before
```
10 → 20 → 30 → NULL
```
### After
```
30 → 20 → 10 → NULL
```
### Program
```c
struct Node* reverse(struct Node *head)
{
    struct Node *prev = NULL;
    struct Node *current = head;
    struct Node *next = NULL;
    while(current != NULL)
    {
        next = current->next;
        current->next = prev;
        prev = current;
        current = next;
    }
    return prev;
}
```
## Time Complexity of Operations
| Operation              | Time Complexity |
| ---------------------- | --------------- |
| Traversal              | O(n)            |
| Insertion at Beginning | O(1)            |
| Insertion at End       | O(n)            |
| Deletion at Beginning  | O(1)            |
| Deletion at End        | O(n)            |
| Searching              | O(n)            |
| Reversing              | O(n)            |
